This project implements an **Extractive Text Summarization system** using classical Natural Language Processing (NLP) techniques.  
Instead of generating new text, the model selects the most important sentences from the original document using **TF-IDF-based sentence ranking**.

The project is built with a **modular Python architecture** and exposed via a **FastAPI REST API**.
